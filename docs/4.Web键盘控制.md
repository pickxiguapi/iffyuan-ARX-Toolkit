# 4. Web 键盘控制

浏览器内键盘遥操作 ARX LIFT2，支持双臂、底盘、升降全部子系统，带三路 D405 实时画面。

## 启动

```bash
# 真机（需先启动 ROS2 节点）
python scripts/web_control_demo.py

# 本地调试（无需硬件）
python scripts/web_control_demo.py --mock

# 自定义端口
python scripts/web_control_demo.py --port 9090
```

打开 `http://localhost:8080` 即可。

## 界面

```
┌─────────────────────────────────────────┐
│  LEFT·D405    HIGH·D405    RIGHT·D405   │  ← 三路相机
├──────────┬──────────┬───────────────────┤
│  左臂     │  右臂     │  升降 / 步长      │  ← 状态面板
│  位姿+关节 │  位姿+关节 │                   │     当前控制臂高亮
│  夹爪进度条 │  夹爪进度条 │                   │
├─────────────────────────────────────────┤
│           键盘控制映射示意               │  ← 支持点击 & 按键
└─────────────────────────────────────────┘
```

## 键盘映射

### 臂控制（Tab 切换左/右臂）

| 按键 | 功能 |  | 按键 | 功能 |
|------|------|--|------|------|
| W/S | Y 前/后 |  | I/K | Roll ± |
| A/D | X 左/右 |  | J/L | Pitch ± |
| Q/E | Z 上/下 |  | U/O | Yaw ± |
| Space | 夹爪开/关切换 |  | Tab | 切换左臂 ↔ 右臂 |

### 底盘（按住移动，松开停车）

| 按键 | 功能 |
|------|------|
| ↑ / ↓ | 前进 / 后退 |
| ← / → | 左移 / 右移 |
| , / . | 左转 / 右转 |

### 升降 & 系统

| 按键 | 功能 |
|------|------|
| = / - | 升 / 降 |
| 1 / 2 / 3 | 步长档位：精细 / 中等 / 粗调 |
| G | 重力补偿模式 开/关 |
| R | 复位（双臂回零 + 夹爪关闭） |
| Esc | 急停 |

## 步长档位

| 档位 | 平移 | 旋转 | 底盘 | 升降 |
|------|------|------|------|------|
| 1 精细 | 0.4 mm | 0.11° | 0.32 m/s | 0.2 |
| 2 中等 | 0.5 mm | 0.17° | 0.40 m/s | 0.3 |
| 3 粗调 | 2.0 mm | 0.57° | 1.20 m/s | 0.8 |

默认档位 1（精细），臂控制按住连续移动（80ms 间隔），底盘松手即停。

## Mock 模式

`--mock` 不连接真机和相机，三路画面显示占位图，所有操作走 `MockARXEnv` 模拟。用于：

- 机器 A 无硬件时调试 UI 布局
- 验证前后端通信逻辑
