# 3. 数据采集

Collector 模块：env + teleop → Zarr 数据集 → LeRobot v3 格式。

## 快速开始

```bash
# 机器 B 上运行（有硬件）

# 1. 主从遥操作采集
python scripts/collect_data.py \
    --dataset datasets/pick_cup.zarr \
    --episodes 50 \
    --teleop leader_follower \
    --leader-side left \
    --hz 30 \
    --cam-mode rgbd \
    --image-size 640 480 \
    --task "pick up the cup"

# 2. 断点续采（自动识别已有 episodes）
python scripts/collect_data.py \
    --dataset datasets/pick_cup.zarr \
    --episodes 50

# 3. 转换为 LeRobot v3 格式
python scripts/convert_to_lerobot_v3.py \
    --zarr datasets/pick_cup.zarr \
    --output lerobot_datasets/pick_cup \
    --repo-id iffyuan/arx_pick_cup \
    --task "pick up the cup" \
    --fps 30 \
    --use-videos
```

## 架构

```
LeaderFollower (后台线程, 50Hz)
    leader_joint → lowpass → deadband → follower
              ↓
Collector (前台, --hz 限速)
    1. obs = env.get_observation()   ← 含相机、关节、末端、升降
    2. action = teleop.last_command  ← 从 teleop 线程读取最新 action
    3. buffer.append(obs, action)
    4. 按键检测: Space=开始, Enter=结束, Ctrl+C=退出
              ↓
    Zarr 数据集 (Blosc zstd 压缩)
              ↓
    convert_to_lerobot_v3.py → LeRobot v3 格式
```

### 键盘交互

采集使用 raw terminal 模式，单键无需回车：

| 按键 | 作用 |
|------|------|
| **Space** | 开始录制当前 episode |
| **Enter** | 结束当前 episode，保存到 Zarr |
| **Ctrl+C** | 退出采集 |

每个 episode 结束后自动打印摘要（步数、FPS、磁盘占用），可随时按 Space 开始下一个。

## Zarr 存储格式

```
dataset.zarr/
├── data/
│   ├── rgb_camera_l         (N, 3, H, W)  uint8     # 3 相机 RGB
│   ├── rgb_camera_h         (N, 3, H, W)  uint8
│   ├── rgb_camera_r         (N, 3, H, W)  uint8
│   ├── depth_camera_l       (N, 1, H, W)  uint16    # [rgbd 模式] 深度
│   ├── depth_camera_h       (N, 1, H, W)  uint16
│   ├── depth_camera_r       (N, 1, H, W)  uint16
│   ├── left_eef_pos         (N, 7)  float32          # 末端位姿
│   ├── left_joint_pos       (N, 7)  float32          # 关节 + gripper
│   ├── right_eef_pos        (N, 7)  float32
│   ├── right_joint_pos      (N, 7)  float32
│   ├── base_height          (N, 1)  float32          # 升降高度
│   ├── action_left          (N, 7)  float32          # 动作：6 joint + gripper
│   ├── action_right         (N, 7)  float32
│   ├── action_base          (N, 3)  float32          # 底盘速度
│   ├── action_lift          (N, 1)  float32          # 升降
│   ├── timestamp            (N,)    float64           # Unix 时间戳
│   └── episode              (N,)    uint16            # episode 编号
└── meta/
    ├── episode_ends         (M,)    uint32            # 各 episode 累计结束索引
    └── config               (JSON attrs)              # 采集参数快照
```

## LeRobot v3 转换

### 特征映射

```
observation.state      = 14D  — left_joint_pos(7) + right_joint_pos(7)
observation.eef_pos    = 14D  — left_eef_pos(7) + right_eef_pos(7)     [可选]
observation.base_state = 1D   — base_height                            [可选]
observation.images.camera_l/h/r — RGB 图像
action                 = 18D  — action_left(7) + action_right(7) + action_base(3) + action_lift(1)
```

### 转换脚本参数

```
--zarr, -i       输入 Zarr 路径（必需）
--output, -o     LeRobot 输出目录
--repo-id        数据集 repo ID（如 iffyuan/arx_pick_cup）
--fps            帧率（默认 30）
--robot-type     机器人类型（默认 arx_lift2）
--task           任务描述
--episodes       只转换前 N 个 episode
--use-videos     视频格式存储图像（推荐大数据集）
--no-eef         不包含 eef_pos 特征
--no-base-state  不包含 base_state 特征
--dry-run        只检查映射逻辑，不执行转换
```

### dry-run 检查

```bash
python scripts/convert_to_lerobot_v3.py \
    --zarr datasets/pick_cup.zarr \
    --dry-run
```

输出所有数组维度、episode 信息、特征映射，不写入任何文件。

## collect_data.py 参数

```
--dataset, -d      Zarr 数据集路径（默认 datasets/demo.zarr）
--episodes, -n     目标 episode 总数，含已有（默认 3）
--teleop           遥操作方式（目前支持 leader_follower）
--leader-side      Leader 臂（left / right，默认 left）
--hz               采集频率 Hz（默认 30）
--cam-mode         rgb 或 rgbd（默认 rgbd）
--image-size W H   图像尺寸（默认 640 480）
--task             任务描述
--save-video       保存每个 episode 的回放 MP4 视频
--control-rate     Teleop 控制频率 Hz（默认 50）
--lowpass-alpha    低通滤波系数（默认 0.5）
--deadband         死区阈值（默认 0.004）
```

## Collector API

```python
from arx_toolkit.env import ARXEnv
from arx_toolkit.teleop import LeaderFollowerTeleop
from arx_toolkit.collect import Collector

env = ARXEnv(action_mode="absolute_joint", camera_type="rgbd")
teleop = LeaderFollowerTeleop(env, leader_side="left")
teleop.start()

collector = Collector(
    env=env,
    action_source=lambda: teleop.last_command,
    dataset_path="datasets/demo.zarr",
    num_episodes=50,
    hz=30.0,
    cam_mode="rgbd",
    image_size=(640, 480),
    task="pick up the cup",
)
collector.run()  # 阻塞式采集循环

teleop.stop()
env.close()
```

### Collector 参数

| 参数 | 类型 | 默认 | 说明 |
|------|------|------|------|
| `env` | ARXEnv | — | 已初始化的环境 |
| `action_source` | `() -> dict` | — | 返回当前 action 的 callable |
| `dataset_path` | str | `datasets/demo.zarr` | Zarr 路径 |
| `num_episodes` | int | 3 | 目标 episode 总数 |
| `hz` | float | 30.0 | 采集频率 |
| `cam_mode` | str | `rgbd` | `rgb` 或 `rgbd` |
| `image_size` | (int, int) | (640, 480) | 图像 (W, H) |
| `task` | str | `""` | 任务描述 |
| `save_video` | bool | False | 保存 MP4 回放 |
| `video_fps` | float | None | 视频帧率（默认 = hz） |

### action_source 说明

Collector 不关心 action 从哪来。`action_source` 是一个无参 callable，返回标准 action dict：

```python
{
    "left":  np.ndarray(7,) | None,
    "right": np.ndarray(7,) | None,
    "base":  np.ndarray(3,) | None,
    "lift":  float | None,
}
```

对于 LeaderFollowerTeleop，直接用 `teleop.last_command`；也可以自定义。
