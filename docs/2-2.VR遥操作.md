# 2-2. VR 遥操作

Quest 3 VR 头显双臂遥操作 ARX LIFT2。

## 快速开始

```bash
# 机器 B 上运行（有硬件）
python scripts/teleop_vr.py

# Quest 3 浏览器打开 https://192.168.101.21:8443 (机器B的IP)
# 首次访问会提示证书不安全 → 点击"继续"
# 点击 "Start Controller Tracking" 进入 AR 模式
# 握住 grip 键移动手柄 → 机械臂末端跟随
# 按 trigger 控制 gripper
# Ctrl+C 退出
```

## 架构

```
Quest 3 浏览器 (A-Frame WebXR)
    │  vr_app.js: 每帧采集手柄 position/quaternion/trigger/grip
    │  通过 WSS 发送 JSON（~60 Hz）
    ▼
VRTeleop (Python)
    ├── HTTPS 服务 :8443（提供 WebXR 页面，WebXR 需要 HTTPS）
    ├── WebSocket 服务 :8442（接收手柄数据）
    ├── 手柄数据解析 → 更新内部状态
    └── 控制循环 20 Hz：
        读最新手柄状态 → delta_eef action → env.step()
```

## 控制映射

| VR 输入 | 机器人动作 |
|---------|-----------|
| Grip 按下 | 激活控制，记录 origin 位置/姿态 |
| Grip 持续按住 + 手柄移动 | `delta_xyz = (current - origin) * scale` |
| Grip 持续按住 + 手柄旋转 | `delta_roll/pitch` 从相对四元数提取 |
| Grip 松开 | 停止发送臂命令（arm action = None） |
| Trigger > 0.5 | Gripper 打开 |
| Trigger ≤ 0.5 | Gripper 关闭 |

左手柄控制左臂，右手柄控制右臂。

**不控制底盘和升降。**

## 命令行参数

```
--https-port   HTTPS 端口（默认 8443）
--ws-port      WebSocket 端口（默认 8442）
--host         绑定地址（默认 0.0.0.0）
--rate          控制循环频率 Hz（默认 20）
--scale        VR 位移 → 机器人位移缩放（默认 1.0）
--rot-scale    旋转缩放（默认 1.0）
--certfile     SSL 证书路径（默认自动生成）
--keyfile      SSL 密钥路径（默认自动生成）
```

## 坐标轴校准

默认使用恒等映射（VR xyz → 机器人 xyz），可能需要在真机上调整：

```python
vr = VRTeleop(
    env,
    axis_mapping=(0, 2, 1),        # VR x→robot x, VR z→robot y, VR y→robot z
    axis_sign=(1.0, -1.0, 1.0),   # 翻转 y 轴
    vr_to_robot_scale=0.5,         # 缩小位移
)
```

## SSL 证书

WebXR 要求 HTTPS。脚本首次运行时自动生成自签名证书（`cert.pem` / `key.pem`）。
Quest 3 浏览器访问时会提示不安全，点击"继续"即可。
