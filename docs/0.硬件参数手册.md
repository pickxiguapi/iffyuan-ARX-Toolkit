# ARX LIFT2 硬件参数手册

> 数据来源：URDF 模型（`src/urdf_lift2/`）、C++/Python SDK（`src/LIFT/`）、实机标定

---

## 1. 系统总览

ARX LIFT2 是一款**双臂移动操作机器人**，由底盘、升降平台、左臂、右臂四个模块组成。

| 参数 | 值 |
|------|-----|
| 系统类型 | 双臂移动操作机器人 |
| 总质量 | ~39.6 kg |
| 通信协议 | CANBus via SocketCAN |
| 控制频率 | 500 Hz（2 ms 周期） |
| 电机在线检测 | 100 ms 超时 |
| 总关节数 | 21（底盘 3 + 升降 1 + 左臂 8 + 右臂 8） |

### 质量分布

| 部件 | 质量 |
|------|------|
| 底座（base_link） | 21.045 kg |
| 底盘轮子 ×3 | 0.575 kg × 3 |
| 升降平台（body4） | 14.181 kg |
| 单臂（含夹爪） | ~3.48 kg |

---

## 2. 底盘

3 个全向轮（Omni wheels）独立驱动，支持全向移动。

| 参数 | 值 |
|------|-----|
| 轮子数量 | 3（前轮 + 后左 + 后右） |
| 关节类型 | Continuous（无限旋转） |
| 单轮扭矩 | 17 N·m |
| 单轮最大转速 | 27 rad/s |
| 速度指令范围 | vx/vy: [-1.5, 1.5]，vz(旋转): [-2.0, 2.0] |

### 关节编号

| URDF 名 | 说明 |
|----------|------|
| joint1 | 前轮驱动 |
| joint2 | 后左轮驱动 |
| joint3 | 后右轮驱动 |

---

## 3. 升降平台

| 参数 | 值 |
|------|-----|

## 4. 单臂关节参数

左臂和右臂**完全对称**，结构镜像安装在升降平台两侧（Y 轴正负方向）。

每臂 = **6 回转关节 + 1 平行夹爪（2 指 prismatic）** = 8 个 URDF 关节。

### 4.1 回转关节

| 关节 | URDF（左/右） | 类型 | 扭矩 | 最大速度 | 下限 (rad) | 上限 (rad) | 说明 |
|------|---------------|------|------|---------|-----------|-----------|------|
| 1 | left_joint11 / right_joint21 | Revolute | 27 N·m | 5.5 rad/s | -2.094 | 3.142 | 基座旋转 |
| 2 | left_joint12 / right_joint22 | Revolute | 27 N·m | 5.5 rad/s | 0.000 | 3.665 | 肩部 |
| 3 | left_joint13 / right_joint23 | Revolute | 27 N·m | 5.5 rad/s | 0.000 | 3.240 | 肘部 |
| 4 | left_joint14 / right_joint24 | Revolute | 7 N·m | 20 rad/s | -1.671 | 1.671 | 腕部 1 |
| 5 | left_joint15 / right_joint25 | Revolute | 7 N·m | 20 rad/s | -1.671 | 1.671 | 腕部 2 |
| 6 | left_joint16 / right_joint26 | Revolute | 7 N·m | 20 rad/s | -2.094 | 2.094 | 腕部 3 |

**扭矩分级：**
- 肩/肘部（joint 1-3）：27 N·m — 标准关节
- 腕部（joint 4-6）：7 N·m — 轻型精细关节

### 4.2 夹爪

| 参数 | 值 |
|------|-----|
| URDF 关节名（左） | left_joint17（开）, left_joint18（闭） |
| URDF 关节名（右） | right_joint27（开）, right_joint28（闭） |
| 关节类型 | Prismatic |
| 最大力 | 300 N·m |
| 最大速度 | 0.053 m/s |
| 开口行程 | 0 ~ 44 mm |
| 软件归一化 | [0, 1]（0 = 全闭，1 = 全开） |
| 硬件原始范围 | [-3.4, 0.0] |

## 5. 控制器关节顺序

YAML 配置中的 21 关节排列：

```
索引 0:  （占位符）
索引 1-3:  joint1, joint2, joint3              — 底盘 3 轮
索引 4:    joint4                               — 升降平台
索引 5-12: left_joint11 ~ left_joint18         — 左臂 6 关节 + 夹爪 2
索引 13-20: right_joint21 ~ right_joint28       — 右臂 6 关节 + 夹爪 2
```

## 6. 电机类型

| 电机类型 | 应用 |
|----------|------|
| MotorType3 (Mtp3Dlc) | 升降电机、腰部电机、头部偏航/俯仰 |
| ChassisType1 | 底盘 3 轮速度控制 |

### 状态反馈

每个关节的 `HybridJointStatus` 包含：

```
position  — 关节角度 (rad) 或线性位置 (m)
velocity  — 速度
torque    — 扭矩/力
current   — 电流
```
